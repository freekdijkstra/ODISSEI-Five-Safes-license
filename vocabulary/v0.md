# TRE Vocabulary

IRI: `https://w3id.org/tre/vocabulary/v0`

Namespace: `https://w3id.org/tre/vocabulary/v0#` (prefix `tre:`)

An ODRL Profile that defines the terms needed to write access policies for Trusted Research Environments (TREs), such as the ODISSEI Five Safes License.

Classes defined in this schema: [TREService](#treservice), [TRE](#tre).

Actions defined in this schema: [analyse](#analyse), [declareIntendedPurpose](#declareintendedpurpose), [provideAccessPurpose](#provideaccesspurpose), [acceptTerms](#acceptterms).

Left Operands defined in this schema: [eduPersonAffiliation](#edupersonaffiliation).

## Classes

### TREService

| Class                  | **tre:TREService** |
|:-----------------------|:-------------------|
| URI                    | `https://w3id.org/tre/vocabulary/v0#TREService` |
| Label                  | TRE Service |
| Description            | A Service that provides a trusted research environment (TRE). |
| Scope note             | Not directly used in ODRL constraints. Use tre:TRE instead. |
| `rdfs:subClassOf`      | `dcat:DataService`, `schema:Service` |
| `schema:serviceOutput` | `tre:TRE` |

### TRE

| Class                    | **tre:TRE** |
|:-------------------------|:------------|
| URI                      | `https://w3id.org/tre/vocabulary/v0#TRE` |
| Label                    | Trusted Research Environment |
| Alt label                | Secure Processing Environment |
| Description              | A controlled compute environment in which permitted analyses on a dataset may be performed, without the data leaving the environment. |
| Scope note               | Instances of TRE are intended to be used as the odrl:rightOperand of a Constraint whose odrl:leftOperand is odrl:virtualLocation. |
| `rdfs:subClassOf`        | `tech:Platform`, `dpv:InformationFlowControl` |
| `schema:potentialAction` | `tre:analyse` |

Note: `tre:TRE` asserts that it does *not* have `schema:potentialAction odrl:distribute`:
The odrl:distribute action is impossible in a TRE.
